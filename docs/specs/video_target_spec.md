# 動画生成仕様書（ターゲット仕様）

本ドキュメントは、YouTubeチャンネル「気合イングリッシュ」の既存動画を分析し、今後の動画自動生成システム（`video_gen.py` 等）が目指すべき「ターゲット仕様」を定義したものです。

## 1. 共通仕様（全動画共通）

全動画に共通する画面レイアウトとして、背景は黒を基調としつつ、微細なノイズテクスチャを加えたデザインを採用します。これは完全なベタ塗り（`#000000`）ではなく、質感を持たせることで長時間の視聴でも目に優しく、かつ単調さを避ける効果を狙ったものです。

また、画面右下には「気合イングリッシュ work hard」のロゴ（脳と力こぶのモチーフ）を固定配置します。このロゴは透明度（アルファ値）を適切に調整し、学習コンテンツの邪魔にならないよう配慮する必要があります。

## 2. Word Audio Mode（英単語聞き流し動画）仕様

対象ジャンル: `vocab_teppeki`, `vocab_target1900`, `vocab_target1400`, `vocab_target1200`, `vocab_leap`

### 2.1 画面レイアウトとテキストデザイン

画面レイアウトは完全な中央揃え（Center Alignment）を基本とします。英単語は画面中央やや上に大きく配置し、その直下に日本語訳を配置します。画面左上には「No. 0001」のような通し番号を小さく表示します。

| 項目 | フォント | 色 | サイズ・配置 | 備考 |
|---|---|---|---|---|
| **英単語** | 太字のサンセリフ体（ゴシック体） | 青色ベース＋白い縁取り | 画面内で最大 / 中央上 | 力強く明瞭な印象を与える |
| **日本語訳** | シンプルなゴシック体 | 白色（`#FFFFFF`） | 英単語より一回り小さい / 中央下 | 複数の意味がある場合は丸数字（①、②等）でリスト化 |
| **通し番号** | ゴシック体 | 白色 | 小さい / 左上 | 進行状況を示す |

### 2.2 シーン構成と演出

動画は5秒間のカウントダウンから始まり、視聴者の集中力を高める演出を行います。その後は「英単語のみ表示」から始まり、設定された間隔を置いて「日本語訳を追加表示」するという学習ループを繰り返します。

テキストの出方は、瞬間表示（カット）または非常に短いフェードインを使用します。派手なアニメーション（スライドやバウンスなど）は一切排し、テンポと学習効率を最優先します。1単語あたりの表示時間は通常2〜4秒程度とし、次々と切り替わるフラッシュカード形式を採用します。

### 2.3 音声とBGM

学習パートでは**BGMを一切使用しません**。これは、視聴者が音声（発音）に完全に集中できる環境を作るためです。

音声は、英単語についてはネイティブスピーカーによるクリアな発音を使用し、日本語訳については日本人ナレーターによる読み上げを行います（日本語音声は設定でON/OFF可能とします）。演出として、単語が表示されるタイミングで「シュッ」という風切り音などのミニマルな効果音を鳴らし、テンポの良さを強調します。カウントダウン時にはビープ音を使用します。

## 3. リスニング動画仕様

対象ジャンル: `listening_todai`, `listening_toeic`, `listening_teppeki`

### 3.1 画面レイアウトとテキストデザイン

テキスト配置は中央配置を基本としますが、長文スクリプトを表示する場合は可読性を確保するために左揃え（Left Alignment）も併用します。また、現在のセクション（Listening, Question, Reviewなど）を画面上部または中央に明記します。

テキストは視認性の高い白のサンセリフ体（ゴシック体）で統一します。Reviewセクションにおいては、重要な英単語やフレーズを**赤色**でハイライトし、学習者の注意を引くデザインとします。

### 3.2 シーン構成と演出

リスニング動画は明確な4部構成を採用します。

1. **タイトル**: 冒頭にタイトルカードを表示します。
2. **Listening Section**: 音声のみを流し、画面は静止画または最小限のテキストのみとします。これにより視覚情報を遮断し、聴覚への集中を促します。
3. **Question Section**: 設問を表示し、読み上げを行います。
4. **Review Section**: スクリプト全文を表示し、音声の進行に合わせてテキストを表示またはハイライトします。

各セクションの冒頭にはタイトルカードを挿入し、進行状況を明確にします。

### 3.3 音声とBGM

ナレーションは、タイトルコールや案内には日本語または英語を使用し、リスニング音声には実際の試験を想定して複数のネイティブスピーカー（男女混声など）を起用します。セクションが切り替わる際には、短いチャイム音などの効果音を挿入し、構成の区切りを聴覚的に伝えます。
